# XMP Test Fixtures

Fixtures in this folder are used by `meta/xmp` parser tests and benchmarks.

## Source provenance

- `acr_sidecar.xmp`: internet-derived sample based on <https://gist.github.com/KelSolaar/6027420>
- `dng_embedded.xmp`: internet-derived sample based on <https://gist.github.com/1024squared/5a66b5bb4fdf2986f0d85b93a52f1dba>
- `lightroom_sidecar.xmp`: local sidecar sample used for coverage and performance regression testing
- `jpeg.xmp`: local JPEG-embedded XMP sample
- `canon_eos_r6_cr3_lightroom_custom_adjustments.xmp`: local Canon EOS R6 CR3 sidecar with non-default Lightroom adjustments
- `canon_eos_r6_cr3_lightroom_blue_label_baseline.xmp`: local Canon EOS R6 CR3 sidecar with Blue label and mostly baseline edits
